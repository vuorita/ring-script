This project is a lightweight web ring for independent artists. Each member has their own website or blog, and this script shows a shared list of members on each participating site.

The member list is stored in a public JSON file on GitHub and updated manually by the maintainer. To join, artists must send their name and website URL to the maintainer and agree to having their info listed publicly.
