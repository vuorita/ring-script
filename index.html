This project is a lightweight web ring script originally developed for the Taidetta.net artist network. It allows independent artists to showcase their own websites or blogs by displaying a shared list of members on every participating site.

The member list is stored in a public JSON file on GitHub and is manually updated by the maintainer. Although the code was created for Taidetta.net, it is freely available for anyone to use.

To join, artists send their name and website URL to the maintainer and agree to have their information listed publicly.
